# World Trade Online Banking: AI Strategy Game

## 📊 Project Overview
This is a high-stakes simulation game where a user competes against a computer-controlled AI to dominate the global market. Players start with a budget of 2,000,000 units and must strategically decide which countries to acquire based on price and remaining capital.

## 🚀 Key Features
- **AI Competitor:** Features an automated opponent that makes independent buying decisions using randomized logic.
- **Dynamic Pricing:** Contains a dataset of 50+ countries with varied economic valuations.
- **Complex Win Conditions:** Winners are determined by a dual-metric system—both total country count and remaining liquidity must exceed the opponent's.

## 🛠️ Technical Stack
- **Language:** Python 3.10+
- **Modules:** `random` (for game mechanics and AI decision-making)
- **Data Structures:** Dictionaries for the global economic database; Sets and Lists for inventory management.
